PvP Game Manager: Bridge Gamemode

Information on how to manipulate the bridge.xml include file developed by zzuf for the open source PGM plugin.

All information is up to date as of include version 2024.03.10-1.

Removing Match Timelimits

As of version 2024.03.10-1, you can remove the timelimit in a match by just running /action trigger remove-timelimit.

Older versions (pre 2024.03.10-1)

To remove timelimits on the previous versions of the include, you must follow these two steps:

1. Run /cancel to cancel all countdowns.
2. Run /action trigger remove-timelimit.
