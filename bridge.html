Removing Match Timelimits

Note: this information will become obsolete once PGM #1260 is merged and the include file is updated.

To remove timelimits on the current version of the include, you must follow these two steps:

1. Run /cancel to cancel all countdowns.
2.Run /action trigger remove-timelimit.
