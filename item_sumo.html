PvP Game Manager: Item Sumo Gamemode

Information on how to manipulate the item-sumo.xml include file developed by zzuf for the open source PGM plugin.

All information is up to date as of include version 2024.03.11-2.

Toggling hills

 You can run  /variable set hill_disabled 1 to disable hills, and
 /variable set hill_disabled 0 to re-enable them.

A message will appear along with a sound effect to let everyone know.

If the blocks below the hill and the value in the map's hill-block-on constant match,
those blocks will get changed to whatever hill-block-off is (and viceversa)

This means that by default, Emerald Blocks will get converted to Coal Blocks.
